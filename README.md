# gpac-build
gpac build script

## Compilation

```
$ ./build-linux.sh
$ ls ~/bin
gpac  ffprobe  ffserver  ...
```

## Easy Install

```
curl -sL https://github.com/kawaz/gpac-build/releases/download/gpac-binary-linux64/gpac-binary-linux64.tar.gz |
  sudo tar xz -C /usr/local/bin --strip=1 --no-same-owner --no-same-permissions
```
